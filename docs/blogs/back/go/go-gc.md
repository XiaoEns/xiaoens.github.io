## Go v1.3 标记清除法
+ 标记：Mark Phase
+ 清除：Sweep Phase

go v1.3 之前步骤：
1. 启动 STW
2. 标记对象
3. 清除对象
4. 停止 STW

go v1.3 步骤
1. 启动 STW
2. 标记对象
3. 停止 STW
4. 清除对象

缺点：
1. STW，让程序暂停，会出现卡顿
2. 标记需要扫描整个 heap
3. 清除数据会出现 heap 碎片

## 屏障机制
+ 强三色不变式: 不允许黑色对象引用白色对象
+ 弱三色不变式：黑色对象可以引用白色对象，但必须有灰色对象引用该白色对象，或者该白色对象的上游有灰色对象

### 插入屏障
在对象 A 引用其他对象时，触发插入屏障，栈空间不会触发插入屏障

### 删除屏障
在对象 A 被删除时，触发删除屏障
