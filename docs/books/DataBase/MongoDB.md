## 介绍
MongoDB是什么？
+ 文档数据库
+ NoSQL类型
+ 由 C++ 编写的开源数据库系统

特点：
+ 面向文档
+ 高性能
+ 高可用
+ 丰富的查询语句
+ 支持主流开发语言
+ 支持索引
